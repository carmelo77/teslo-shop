# Teslo API

1. Clonar proyecto

2. ```Npm install```

3. Llenar variables de entorno en .env

4. Definir variables de entorno

5. Levantar base de datos
```
docker-compose up -d
```

6. Levantar servidor: ```npm run start:dev```
